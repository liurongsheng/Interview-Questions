# 可响应对象

属性可以被监听的

- 访问器 get/set，简单

```js
  class {
    get 名字(){

    }
    set 名字(val){

    }
  }
```

- defineProperty，缺陷：操作数组内部的东西——无法响应

  字面意思：定义一个属性

  - 数组、json 监听不完整

  Object.defineProperty(obj, 名, {...});

- Proxy，支持的类型多并且没有缺陷，好用
  
  - json
  - 函数
  - 数组
  - 类
